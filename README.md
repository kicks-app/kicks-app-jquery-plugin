jquery-plugin
=============

> JQuery Plugin Boilerplate

Usage
-----

Include dependencies

```html
<script src="https://code.jquery.com/jquery-1.11.2.min.js"></script>
<script src="dist/jquery.plugin.min.js"></script>
```

Provide markup

```html
<div class="plugin"> </div>
```

Initialize Plugin

```js
$('.plugin').plugin();
```
